linux下设置

常用的event.button事件

  • IE下
    • 没有按键动作的时候 window.event.button = 0
    • 左键是 window.event.button = 1
    • 中键是 window.event.button = 4
    • 右键是 window.event.button = 2
  • Firefox下
    • 没有按键动作的时候 event.button = 0
    • 左键是 event.button = 0
    • 中键是 event.button = 1
    • 右键是 event.button = 2
  • 通过修改_MouseGesture2.uc.js，可以实现中键手势
