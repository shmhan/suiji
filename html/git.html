.git/config文件

github的ssh登录需要修改git目录下的config文件如下： [remote "origin"] url = https://github.com/shmhan/suiji.git fetch = +refs/heads/:refs/remotes/origin/ [branch "master"] remote = origin merge = refs/heads/master 修改后如下： [remote "origin"] url = git@github.com:shmhan/suiji.git fetch = +refs/heads/:refs/remotes/origin/ [branch "master"] remote = origin merge = refs/heads/master

github的远程改动

git远程库改动后，应该首先在本地git pull，然后git commit -a。 不然的话，中间夹杂各种快照的git库会让你彻底崩溃。
